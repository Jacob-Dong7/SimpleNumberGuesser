<h1>Simple Number Guesser Program</h1>

<h2>Description</h2>
This is a simple number guessing program. The user will be prompted to enter a range for which the random number will be generated in, the user will then attempt to guess the number and the program will provide feedback each guess to assist the user in finding the random number.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>C++</b>
- <b>Visual Studio Code</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
